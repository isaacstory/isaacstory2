## Special Markers

Preserve these suffixes:
- `_ALTERADO` - Indicates abnormal/flagged results
- `_INVALIDO` - Indicates invalid/superseded document
- `_EN` / `_PT` - Language variants (English/Portuguese)
- `_1de4`, `_2de4` - Multi-part documents
